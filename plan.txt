 - Log in

 - Continue as a guest

 ---ADMIN ACCOUNT---
 The admin account can edit, view and do the following things: 
 - Edit fully the calendar. The calendar has a default weekly schedule, which only when edited changes for the future. So it's as easiest as possible. 
 - The admin can deactivate, edit, add and remove events/classes from the calendar. 
 - The name (accounts) of the students can be selected for the lessons. Personalization of the calendar with a color palette. 
 - Adding events in a calendar:
  * Student account
  * Date 
  * Start time - End time in format 0-24h. European
  * Online or in-person 
  * Small palette with an example of the calendar view 

  * Labeling an event in the calendar view 

  * Deactivating days of the calendar

  Adding an event: 
  * Event title
  * Event date (optional)
  * Event description - with files, images...
  - Editing an event

  When posting an event it should be posted in the social networks of the course. 

 ---STUDENT ACCOUNT---
 The students can view the whole calendar, events and soon maybe play language games.
